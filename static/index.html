    대본·이미지·영상을 AI에 맡길지, 직접 제공할지 골라주세요.

    🤖

    AI가 모두 생성

    주제만 입력하면 제목·나레이션·이미지·영상까지 AI가 자동으로 만들어줍니다.

    ✍️

    내가 직접 제공

    이미 가지고 있는 대본을 입력하고, 줄마다 이미지·영상을 올리거나 AI에 맡길 수 있어요.

    1. 주제 설정

    정보성: 제품/브랜드 언급 없이 순수 정보 전달 (성분 설명은 가능, 구매 권유 금지)
    홍보성: 네이버 클립 쇼츠용 — 성분/역할 위주로 설명하고 마지막에 블로그 유입 유도
    홍보성 (고정댓글 유도형): 본문은 문제·호기심만, 해답은 고정 댓글로 유도 (주제만 입력, 5줄 반말)

    내 화장품이 해결하는 피부 고민을 적어주세요.
    예: 홍조 피부 진정 방법 / 건조 피부 보습 루틴 / 모공 축소 관리법

    내 화장품을 써야 하는 사람의 증상을 적어주세요.
    예: 얼굴 빨개짐 / 울긋불긋 / 건조한 피부 / 화장이 들뜨고 각질 일어남

    제품의 대표 성분이나, 제품이 문제를 어떻게 해결하는지 적어주세요.
    예시 (성분): 시카(병풀추출물) / 히알루론산 / 나이아신아마이드
    예시 (역할): 샴푸가 노폐물을 자극 없이 제거하고 헤어토닉이 수분을 채워줌
    예시 (혼합): 자연유래성분이 속건조를 해결해서 가려움 해소

    업로드한 제품 중 하나를 클릭해서 선택하세요. 9:16 비율 이미지 권장.

    +
    제품 추가

    주제를 특정 방향으로 좁혀줍니다. 비워두면 AI가 자유롭게 선택해요.
    성분·원리·개념만 입력하세요. 제품명·브랜드명은 쓰면 안 됩니다 (정보성 영상은 제품 언급 금지).
    예: 주제 "탈모 해결법" + 키워드 "맥주효모" → "탈모에 맥주효모가 효과적인 이유" 방향으로 생성
    ※ 제목 생성 후 수정하면 나레이션 방향이 어긋날 수 있어요
